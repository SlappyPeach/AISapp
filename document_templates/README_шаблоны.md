# Шаблоны документов для AISapp

Поля заполнения обозначены как `{{FIELD_NAME}}`. Табличные строки с `LINE_*`, `ITEM_*`, `MATERIAL_*`, `PPE_*` предполагают клонирование строки при генерации документа из списка позиций AISapp.

## Базовые группы полей

- Договоры: `CONTRACT_NUMBER`, `CONTRACT_DATE`, `CUSTOMER_*`, `CONTRACTOR_*`, `SUPPLIER_*`, `BUYER_*`, `OBJECT_*`, `AMOUNT`, `VAT_*`, `WORK_*`.
- Первичные документы: `DOCUMENT_NUMBER`, `DOCUMENT_DATE`, `ORGANIZATION_*`, `SUPPLIER_*`, `BUYER_*`, `WAREHOUSE`, `BASIS_DOCUMENT`.
- Табличные позиции: `LINE_NO`, `MATERIAL_CODE`, `MATERIAL_NAME`, `ITEM_NAME`, `UNIT`, `QUANTITY`, `PRICE`, `LINE_AMOUNT`, `VAT_AMOUNT`, `TOTAL_*`.
- СИЗ: `WORKER_NAME`, `EMPLOYEE_NUMBER`, `PPE_NAME`, `PPE_CODE`, `SERVICE_LIFE_MONTHS`, `START_DATE`.
- Подписи: `DIRECTOR_NAME`, `SITE_MANAGER_NAME`, `RESPONSIBLE_PERSON_NAME`, `LEFT_SIGNER_NAME`, `RIGHT_SIGNER_NAME`.
